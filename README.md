# Currency Translation App

A simple **currency exchange application** built with React.  
The app converts currencies using **real-time exchange rates** fetched from FreeCurrencyAPI.

## Features

- Real-time currency conversion
- Supported currencies: USD, EUR, TRY
- Clean and user-friendly interface
- Input validation for safe calculations
- React Hooks (useState, useEffect)

## Technologies

- React
- Axios
- CSS
- FreeCurrencyAPI
